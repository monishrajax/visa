Modern, production-ready marketing + demo UI built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, **Lucide**, and **Recharts**.

## Tech

- Next.js (requested: v16)
- React + TypeScript
- Tailwind CSS (dark + light mode)
- Framer Motion (subtle motion)
- Lucide icons
- Recharts (demo charts)

## Run locally

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

## Theme

Theme is controlled by a small built-in toggle (light/dark/system) stored in `localStorage` and applied via a `dark` class on the `<html>` element.
